# 6. 월간 건강 보고서

## 6.1 보고서 구성
### 6.1.1 종합 건강 상태
1. 월간 위험도 평균 및 추이
2. 주요 발견 증상 요약
3. 약물 복용 순응도
4. 전화 응답률
5. 전반적 건강 상태 평가

### 6.1.2 상세 분석
- 증상별 상세 분석 (발생 빈도, 심각도 변화, 계절적 요인, 생활습관과의 연관성)
- 권장사항 (의료진 상담 권유, 생활습관 개선 제안, 주의 관찰 증상, 다음 달 모니터링 중점 사항)

## 6.2 의료진 연계 정보
- 구조화된 증상 기록
- 시간대별 증상 발생 패턴
- 환자 주관적 증상 설명
- 위험도 변화 추이

활용 방안:
- 진료 시 참고 자료
- 약물 효과 모니터링
- 질병 진행 상황 파악
- 치료 계획 수립 지원
